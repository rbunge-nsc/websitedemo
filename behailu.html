Common Employment Criteria for these Roles

When deciding on which role or roles to learn in this project, it was helpful to have a cursory understanding of what the job may entail. A search through Indeed.com produced several results and the commonality of these results is below. While this is not an exhaustive list of the requirements each role might have, it gives a beginner an idea on what skills to learn.

Software Development

* Design and develop software * Build websites from requirements and mockups * Use best practices and design standards * Develop unit testing * Focus on the impact to other teams and plan your code accordingly * Prepare for continuous updates and testing * Validate that code meets requirements through code reviews * Use source control management system * Document solutions to issues * Update end user documentations

Testing

* Design, plan and implement automated tests * Document testing procedures * White and black box testing? * Find ways to automate testing

Deployment

* Maintain mirroring of staging to Main * Maintain backups * Design, Build, document and Maintain a scalable solution * Assist other teams in finding issues * Set best practices in team workflow * AWS experience * Responsible for data restoration * Documents issues and resolutions
